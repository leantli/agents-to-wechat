# agents-to-wechat

English summary for the Chinese-first README.

`agents-to-wechat` bridges Codex with WeChat private chats through a Node.js CLI.

## Quick Start

```bash
npm install -g agents-to-wechat
npx agents-to-wechat start
```

## Notes

- Node.js 22.12+ LTS or 24+ is required
- `codex` must be installed and available on `PATH`
- The current release is a `0.x` preview
- Only text messages and WeChat private chats are supported

See [README.md](README.md) for the full Chinese documentation.
